# 🤖 RAG PDF Chatbot

An AI-powered Retrieval Augmented Generation (RAG) application that allows users to upload one or more PDF documents and ask natural language questions about their contents.

Built using LangChain, FAISS, Sentence Transformers, Groq Llama 3, and Streamlit.

---

## 🚀 Features

* Multi-PDF Upload
* Semantic Search
* FAISS Vector Database
* Conversation History
* Source Citations
* Download Chat History
* Groq Llama 3 Integration
* Streamlit Dashboard

---

## 🛠 Tech Stack

* Python
* Streamlit
* LangChain
* FAISS
* Sentence Transformers
* Groq API
* PyPDF

---

## ⚙️ How It Works

PDF Upload
↓
Text Extraction
↓
Chunking
↓
Embeddings Generation
↓
FAISS Vector Store
↓
Semantic Retrieval
↓
Groq Llama 3
↓
Answer Generation
